This plugin helps your SuperOffice CRM integration -which helps you to transfer contact details to WordPress- to collect your site's user roles.


== Description ==

Did you create custom roles in your site? Install and enable this plugin if you want to use these custom roles while creating users in WordPress from SuperOffice CRM.

Once you have installed this plugin, head over to [https://wordpress.superofficeapps.com](https://wordpress.superofficeapps.com/) to link your Wordpress site to your SuperOffice CRM.
With the free SuperOffice for WordPress app, you can import contact details to create or update WordPress user accounts and give your customers, suppliers, students or prospects access to role based content, your e-commerce store or your
e-learning portal. Create a SuperOffice selection, select the WordPress role you want to asign the users to and start the transfer. 

Ready to set things up? Visit our [knowledge base](https://infobridgekb.atlassian.net/wiki/spaces/SAS/pages/1991901218/WordPress). 


Key Features of this plugin

* Lets you use the custom roles while creating users from SuperOffice CRM.


Key Features of the free SuperOffice for WordPress app

* Import SuperOffice CRM selection members directly into WordPress user accounts.
* Select the role your contacts need in WordPress from the WordPress roles.
* SuperOffice CRM and WordPress user-defined fields supported.


== Frequently Asked Questions ==

= Is this plugin mandatory for the SuperOffice app to work? =

No, this plugin makes sure not only the default WordPress roles are shown in the SuperOffice panel, but also the roles that are created by WordPress administrators or plugins.

= Is this plugin of any use without the SuperOffice app? =

No, without the SuperOffice app, there is no need to install this plugin.

= How do I get started? =

Start by installing this plugin if you have custom Wordpress roles.
Then visit [https://wordpress.superofficeapps.com](https://wordpress.superofficeapps.com/) to set up your integration free of charge.


== Screenshots ==
